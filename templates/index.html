Hallo python flask!!!

selamat datang di website flask pada python, kali ini halaman ini ditulis menggunakan templating

pada halaman ini ditambahkan nilaiku yaitu : {{var1}}

tambahan pada halaman ini adalah terdapat nama-nama hari dari parsing python, yaitu:

{% for hariku in var2%}
  • hari {{hariku}}
    • {% endfor %}

    tambahan halaman ini adalah kondisi jika hari minggu berarti:

    {%if var3 == "minggu"%}

    hari libur

    {%else%}

    hari kerja

    {%endif%}

    tambahan halaman ini adalah penggabungan perulangan dan kondisi yaitu bahwa:

    {% for hariku in var2%} {%if hariku == "minggu"%}

    hari {{hariku}} adalah juga hari libur

    {%elif hariku == "sabtu"%}

    hari {{hariku}} adalah hari libur

    {%else%}

    hari {{hariku}} adalah hari kerja

    {%endif%} {% endfor %}
